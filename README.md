**BurpHeaderHelper**是一款Burpsuite插件，主要用于简化和解决Burpsuite对Http的一些操作.

目前实现: `HttpHeader增删改` `HttpCookie` `随机UserAgent` `RepeaterResponse自动解码` `丢弃特定数据包`
